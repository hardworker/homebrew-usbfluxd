# Hardworker Usbfluxd

## How do I install these formulae?

`brew install hardworker/usbfluxd/<formula>`

Or `brew tap hardworker/usbfluxd` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "hardworker/usbfluxd"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
